# react-avapi

### install

```
npm i --save avuxi/react-avapi
```

### usage

```js
import { Map, Lse } from 'react-avapi'

```

```html
<Map confId='5ae6df87a93f4e1e02a6122a'
     lat='41.38605'
     lng='2.17014'
     zoom='13' />
     
<Lse confId='5ae6e03da93f4e1e02a6122b'
     lat='41.390847'
     lng='2.16902' />
```
